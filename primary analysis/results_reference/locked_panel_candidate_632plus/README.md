# Locked-panel candidate-estimator screening

This directory contains aggregate results from 2,000 paired class-stratified
bootstrap `.632+` resamples generated by:

```bash
python "code/models/locked_panel_candidate_632plus.py" \
  --n-bootstrap 2000 \
  --jobs 6
```

The editable manuscript Figure 4 is stored in `../manuscript_figures/`.

The six estimators use the same locked seven-feature radiomics panel and paired
bootstrap samples. Hyperparameters are selected by 3-fold stratified
cross-validation and then held fixed. Strict nested STABL is reported
separately for feature-selection sensitivity.
